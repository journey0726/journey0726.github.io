# 财富

- 创造财富
  - 没有弄清楚应该做什么之前，不应该过分盲目地努力。
  - 财富就是在睡觉时仍在赚钱的资产，金钱是转换时间和财富的手段，身份是社会等级体系所处的位置。
  - 你无法通过廉价的出租自己的时间而致富，你必须拥有产权(如股权)。
  - 向社会提供有价值的东西————他们不知道如何获取，而且要大规模的应用。
  - 选择能和长期合作伙伴长期做的事业。
  - 复利。无论是财富，人际关系还是知识。
  - 学会销售，学会创建。
  - 特殊知识(不能通过培训而得到的知识，不可替代的价值)，杠杆，责任感。
  - 学习微观经济学，博弈论，心理学，洗脑术，伦理学和计算机科学。
  - 读比听快，做比看快。
  - 总结：产品化自己
